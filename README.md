# 百词百科

百词百科是一款在线教育/信息查询类目小程序，通过背后的随机推荐算法，给用户带来清新脱俗的学习和浏览体验，可应用与如下领域：

- 学习外语单词
- 刷应试题
- 浏览成语/歇后语
- 浏览国家颁布的法律条文
